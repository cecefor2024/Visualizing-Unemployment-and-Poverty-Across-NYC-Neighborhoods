Overview
This project analyzes unemployment and poverty rates across NYC boroughs using publicly available socioeconomic data.  
The goal is to identify trends and correlations between population density, poverty, and unemployment levels.

Process
1. Data Collection:  
   - Downloaded data from NYC Open Data on borough-level unemployment and poverty rates.  

2. Data Cleaning (Excel):  
   - Removed duplicates and standardized borough names  
   - Added calculated columns for population density and poverty ratio  

3. Visualization (Tableau):  
   - Created a map showing borough-level unemployment and poverty  
   - Designed scatterplots comparing poverty vs. unemployment  

Key Findings
- The Bronx has the highest poverty and unemployment rates across all boroughs.  
- Lower poverty correlates with higher employment and population density.  
- Manhattan and Queens display more balanced socioeconomic conditions.
